# Project_pet_mixture_moments
Учебный проект по анализу данных. Данный мини проект посвящён оценке параметров смеси двух распределений Пуассона на основе наблюдаемых частот значений случайной величины. Задача мотивирована экспериментальными данными о числе зарегистрированных распадов ядер при нейтронной бомбардировке.
